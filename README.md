# Создание автоматической системы распознавания автомобильных номеров в Гонконге: выявление особенностей и повышение точности распознавания

## Стек:Python, OpenCV, GroundingDINO, YOLOv5, Tesseract OCR, Pandas, Numpy, Matplotlib, Seaborn, Re

Описание: Данный проект направлен на создание автоматической системы распознавания автомобильных номеров на основе изображений транспортных средств, сделанных в условиях городской среды Гонконга. В рамках проекта разработана пайплайн-обработки изображений, включающая детекцию номеров, их локализацию и посимвольное распознавание.

Для решения задачи использовались современные модели детекции объектов, такие как YOLOv8 и GroundingDINO, обученные на специальных датасетах с учетом особенностей номеров Гонконга, а также алгоритмы OCR для точного чтения символов.

Данные для обучения и тестирования собирались как из открытых источников, так и с помощью ручной разметки, включающей координаты боксов номеров и посимвольную разметку.

На этапе анализа проводился сравнительный анализ различных методов и моделей по точности распознавания и скорости обработки. Также осуществлялся анализ ошибок и выявление наиболее частых причин некорректных результатов.

Цель проекта:

Создать эффективную автоматическую систему распознавания автомобильных номеров, которая позволит:

- Детектировать все номера на изображениях транспортных средств в различных условиях освещения и ракурса.
- Распознавать посимвольно номера с высокой точностью, включая номера с нестандартными надписями.
- Обеспечить работу системы в реальном времени для использования в системах автоматической парковки и видеонаблюдения.
- Анализировать особенности номеров в Гонконге, такие как наличие различных форматов, спецсимволов и индивидуальных надписей.

Данные:
Используемые данные включают:

- Изображения транспортных средств, полученные с камер видеонаблюдения.
- Разметка номеров (координаты бокс и посимвольная разметка для обучения OCR-моделей).

Результаты исследования:

- Разработан пайплайн, объединяющий детекцию и распознавание номеров.
- В ходе тестирования достигнута высокая точность обнаружения номеров — 0.85 по метрике mAP.
- Точность распознавания символов — около 0.75 CRR, с учетом различных условий съемки.
- Обнаружены наиболее сложные случаи — номера с нестандартными надписями и поврежденные бока.
- Проведен анализ ошибок, предложены пути повышения точности (например, расширение обучающей выборки, улучшение предобработки изображений).

Это решение позволяет интегрировать систему в реальные условия Гонконга и использовать её для автоматизации контроля транспортных средств на парковках, видеонаблюдения и других сценариях.
